## GitHub Changelog で見た重要更新

- [Kimi K2.7 Code is generally available in GitHub Copilot](https://github.blog/changelog/2026-07-01-kimi-k2-7-is-now-available-in-github-copilot): GitHub Copilot に Kimi K2.7 Code が一般提供され、モデル選択肢が増えました。コスト重視の利用や別モデルとの比較をしやすくなり、既存ユーザーにとっては Copilot の使い分けが広がる更新です。
