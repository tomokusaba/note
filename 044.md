## GitHub Changelog（既存ユーザーにインパクトが強い重要更新）

1. [Copilot CLI: New terminal interface is generally available](https://github.blog/changelog/2026-06-23-copilot-cli-new-terminal-interface-is-generally-available)
   - Microsoft Build 2026 でプレビューされていた **GitHub Copilot CLI の新ターミナルインターフェイスが GA** になりました。
   - タブ型レイアウトを採用し、Issues・Pull Requests・Gists をターミナルから直接閲覧・操作できます。<kbd>c</kbd> キーで参照をプロンプトに挿入し、そのまま Copilot に調査・修正を依頼できます。
   - `/mcp add`・`/skills`・`/plugin`・`/settings` などのコマンドで設定ファイルを手編集せずに MCP サーバーやスキルを管理できます。
   - テーマ対応・スクリーンリーダー対応など、アクセシビリティも強化されています。`copilot update` で更新可能。

2. [GitHub Copilot app support for BYOK](https://github.blog/changelog/2026-06-23-github-copilot-app-support-for-byok)
   - GitHub Copilot アプリが **BYOK（Bring Your Own Key）に対応**。OpenAI・Azure OpenAI・Microsoft Foundry・Anthropic・LM Studio・Ollama などの自社モデルプロバイダーをセッションごとに選択できるようになりました。
   - API キーはローカル OS キーチェーンに保存され、UI に読み返されることはありません。
   - 社内ゲートウェイ経由でのトラフィックルーティングにも対応し、データ境界要件が厳しいエンタープライズ環境での利用が可能になります。

3. [Automatic Dependabot access to GitHub-hosted registries](https://github.blog/changelog/2026-06-23-automatic-dependabot-access-to-github-hosted-registries)
   - Dependabot が **PAT なしでプライベートな GitHub Packages レジストリを読めるように**なりました。
   - パッケージ設定の「Manage Actions access」でリポジトリに Read アクセスを付与するだけで有効になります。`dependabot.yml` の変更は不要で、PAT ベースのレジストリ設定を削除できます。
   - GitHub Packages がサポートするすべてのエコシステムで利用可能。

4. [Fetch Code Quality findings via REST API](https://github.blog/changelog/2026-06-23-fetch-code-quality-findings-via-rest-api)
   - CodeQL の Code Quality 結果を取得する **REST API が Public Preview** になりました。
   - `GET /repos/{owner}/{repo}/code-quality/findings` と `GET /repos/{owner}/{repo}/code-quality/findings/{finding_number}` の 2 つのエンドポイントが追加され、フィルタリング・ページネーションに対応。
   - ツール連携やエージェントによる自動修正ワークフローの構築に活用できます。

5. [Deprecation of Python 3.9 for Dependabot](https://github.blog/changelog/2026-06-23-deprecation-of-python-3-9-for-dependabot)
   - Dependabot が **Python 3.9 のサポートを終了**しました（Python 3.9 は EOL）。
   - Python 3.9 を使い続けると Dependabot が依存更新 PR を作成しなくなる可能性があります。サポート対象の Python バージョン（2026年5月時点では最新 3.14）への移行を推奨。
