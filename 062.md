## GitHub Changelog で見た重要更新

- [Issue fields are now generally available](https://github.blog/changelog/2026-07-02-issue-fields-are-now-generally-available): GitHub の Issue fields が一般提供され、優先度・工数・日付などの構造化メタデータを組織単位で一貫して扱えるようになりました。リポジトリの課題一覧にも表示されるため、運用やレポート作成の負担を下げる更新です。
- [Secret scanning public monitoring for enterprises](https://github.blog/changelog/2026-07-01-secret-scanning-public-monitoring-for-enterprises): 企業向けに、公開されている GitHub 上の漏えいシークレットをリアルタイムで検知して企業へ紐づける機能が公開プレビューになりました。個人フォークや公開 issue からの漏えいも追えるため、既存のシークレットスキャンを補完する意味合いが大きいです。
- [Cost centers now support AI credit pools](https://github.blog/changelog/2026-07-02-cost-centers-now-support-included-usage-caps): Copilot の included AI credits を cost center 単位で上限管理できるようになりました。企業のライセンス負担と利用の境界をより明確にしやすく、既存ユーザーにとっては支出管理の制御が強化される更新です。
