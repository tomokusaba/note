# 2026-07-08 開発者ブログ・GitHub Changelog まとめ

## はじめに

Microsoft Developer Blogs と GitHub Changelog から、直近の更新を整理しました。Microsoft Developer Blogs では .NET に関心がありそうな新規投稿は見つからなかったため、ここでは該当なしとして記載しています。GitHub 側は、既存ユーザーに影響が大きいと思われる更新を中心にまとめています。

## Microsoft Developer Blogs

- **.NET に関心がありそうな新規記事は見つかりませんでした。**
- 直近で確認できた投稿は [Don’t rewrite your CLI for agents](https://devblogs.microsoft.com/blog/dont-rewrite-your-cli-for-agents?WT.mc_id=DT-MVP-5004827) でしたが、内容は CLI とエージェント相性の検証に関するもので、.NET 直結の新着記事ではありませんでした。

## GitHub Changelog

- **Per-user budgets for cost centers in the billing UI**: Enterprise 管理者が、請求 UI からコストセンターごとのユーザー単位の予算を設定できるようになりました。Copilot の利用ベース課金や大規模組織でのコスト管理に直結する変更です。
- **Secret scanning extended metadata and multipart validation**: シークレットスキャンの結果に、所有者や有効期限などの追加メタデータが表示されるようになりました。漏えい対応の優先度付けやトリアージに役立ちます。
- **Restrict who can dismiss reviews in rulesets**: リポジトリ ruleset で、レビューを却下できる対象を制限できるようになりました。ブランチ保護の運用ルールをさらに厳密にできます。
- **GitHub Copilot app available to all**: GitHub Copilot app がすべての Copilot プランで利用できるようになりました。デスクトップからのエージェント駆動開発体験が広がります。
- **Copilot Billing Preview app will be retired on August 3**: Copilot Billing Preview app は 2026 年 8 月 3 日に終了予定です。請求設定内の標準機能へ移行する流れが明確になっています。

## まとめ

- Microsoft Developer Blogs では、.NET 直結の新規記事は確認できませんでした。
- GitHub Changelog では、特にコスト管理・シークレットスキャン・ruleset の運用制御が、既存ユーザーにとって重要な更新でした。
