# 2026-07-10 開発者ブログ・GitHub Changelog まとめ

## Microsoft Developer Blogs

- **Modernize .NET applications in the GitHub Copilot app**（2026-07-09）: GitHub Copilot app から .NET アップグレードを追跡できるようになり、評価・計画・実装タスク・実行・ビルド失敗の確認を 1 つの画面で進められるようになりました。モダナイゼーションの流れを把握しやすくなります。
- 参考: [Modernize .NET applications in the GitHub Copilot app](https://devblogs.microsoft.com/dotnet/modernize-dotnet-in-github-copilot-app/?WT.mc_id=DT-MVP-5004827)

## GitHub Changelog

- **OpenAI's GPT-5.6 Sol, Terra, and Luna are now available in GitHub Copilot**（2026-07-09）: GitHub Copilot に GPT-5.6 系モデルが順次追加されました。Sol は高い推論性能、Terra はバランス型、Luna は軽量・低コストで、用途に応じてモデルを選びやすくなります。
- **Ask Copilot for a repository overview**（2026-07-09）: 初めて見るリポジトリのトップページで、Copilot が目的・技術スタック・貢献ガイドラインを要約して示すようになりました。初見の理解が速くなります。
- **Organization-level targeting for GitHub Code Quality**（2026-07-09）: Organization 管理者が対象リポジトリを絞って Code Quality の有効化・無効化を行えるようになり、導入範囲の制御がしやすくなりました。

## まとめ

- .NET では、GitHub Copilot app から .NET のモダナイゼーションを進める体験が強化されました。
- GitHub では、Copilot のモデル選択肢の拡大と、初見のリポジトリ理解を支援する機能が目立ちます。
