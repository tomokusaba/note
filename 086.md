## GitHub Changelog

### Claude Opus 5 が GitHub Copilot で利用可能に

[Claude Opus 5](https://github.blog/changelog/2026-07-24-claude-opus-5-is-now-available-in-github-copilot) が GitHub Copilot のモデルとして追加されました。複雑で長時間にわたるコーディング作業、複数ツールを使う作業、慎重な推論が必要なタスクを主な対象にしています。

Copilot Pro+、Max、Business、Enterprise の利用者は、Visual Studio Code、Visual Studio、Copilot CLI、Copilot cloud agent、GitHub Copilot app、github.com、GitHub Mobile、JetBrains、Xcode、Eclipse のモデルピッカーから選択できます。段階的なロールアウトのため、すぐに表示されない場合があります。

Business と Enterprise では、管理者が Copilot の設定で Claude Opus 5 のポリシーを有効にする必要があります。対象プランを利用しているチームは、管理設定を確認したうえで、長時間の自律的な実装や検証を伴うタスクで試す価値があります。

Claude Opus 5 は、従量課金でプロバイダーの API リスト価格が適用されます。利用前に [モデルとリクエストの料金](https://docs.github.com/copilot/reference/copilot-billing/models-and-pricing) を確認しておくと安心です。また、高リスクなサイバー領域向けの安全対策により、セキュリティに近い一部の依頼がブロックされる場合があります。

## まとめ

今回の重要な更新は、GitHub Copilot における Claude Opus 5 の提供開始です。対象プランの利用者は、利用環境への段階的な反映、組織ポリシー、従量課金の扱いを確認しておくとよさそうです。
