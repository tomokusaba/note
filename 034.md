### 1) GitHub Code Quality が 2026-07-20 に GA / 有料化予定

- [GitHub Code Quality generally available July 20, 2026](https://github.blog/changelog/2026-06-16-github-code-quality-generally-available-july-20-2026/)

Code Quality の正式提供と課金開始予定が明示され、運用中の組織には影響が大きい更新です。

主なインパクトは次のとおりです。

1. **7/20 以降は有料前提の運用設計が必要**
2. AI 機能の従量課金と CodeQL 実行（Actions minutes）を含めたコスト見積もりが必要
3. 組織全体展開・ダッシュボード・API など、運用面の機能拡張が前提になる

### 2) GitHub Code Quality の組織単位有効化（Public Preview）
