## GitHub Changelog

### CodeQL 2.26.0 adds Kotlin 2.4.0 support and AI prompt injection detection

- GitHub の code scanning で使われる CodeQL が 2.26.0 になりました。github.com の code scanning 利用者には新バージョンが自動デプロイされるため、設定変更なしで解析結果に影響し得る更新です。
- Kotlin は 2.4.0 までサポートされます。C# では Razor Pages の `OnGet`、`OnPost`、`OnPostAsync` などのハンドラーパラメーターが remote flow source として扱われ、`PageModel` 内の `cs/sql-injection` などで検出できる範囲が広がります。
- JavaScript / TypeScript では、信頼できない入力が AI モデルの system prompt に流れ込むことを検出する `js/system-prompt-injection` クエリが追加されました。OpenAI、Anthropic、Google GenAI SDK の追加 API も prompt injection の sink としてモデル化されています。
- 既存の code scanning を運用している場合は、次回の解析で新規アラートや結果の変化がないか確認するとよさそうです。GitHub Enterprise Server では、将来のリリースに含まれる予定です。
- 参考: [CodeQL 2.26.0 adds Kotlin 2.4.0 support and AI prompt injection detection](https://github.blog/changelog/2026-07-10-codeql-2-26-0-adds-kotlin-2-4-0-support-and-ai-prompt-injection-detection/)
