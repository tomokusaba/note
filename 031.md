## GitHub Changelog（既存ユーザーへの影響が大きいもの）

### Copilot code review が組織単位で扱いやすくなった

- [Copilot code review: New configurations and controls](https://github.blog/changelog/2026-06-12-copilot-code-review-new-configurations-and-controls/)

この更新は、**Copilot code review を使っている組織の運用をかなり楽にする** ものです。  
主なポイントは 3 つです。

1. **organization 単位で runner type を設定** できる  
   リポジトリごとの個別設定を減らせます。
2. **Copilot content exclusion** に対応した  
   除外したいファイルやディレクトリをレビュー対象から外せます。
3. **custom instructions の 4000 文字制限を撤廃**  
   `copilot-instructions.md` や `*.instructions.md` を長めに書いても途中で切られません。

公開時刻は **2026-06-12 14:37 PDT** なので、今回の 24 時間枠に入っています。  
個人利用よりも、**組織で Copilot を運用しているチームほど効く更新** です。
